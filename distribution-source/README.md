Эта папка копируется в офлайн-сборку клиента перед упаковкой релиза.

Что сюда можно положить:
- `mods/` — моды NeoForge для клиента.
- `config/` — клиентские конфиги модов.
- `defaultconfigs/` — набор дефолтных конфигов для мира.
- `resourcepacks/` — ресурспаки.
- `shaderpacks/` — шейдеры.

Схема работы:
1. Добавьте или обновите моды вручную.
2. При необходимости поправьте `content/launcher-content.json` и `launcher.config.json`.
3. Выполните `npm run prepare:distribution`.
4. Затем соберите релиз командой `npm run dist`.
